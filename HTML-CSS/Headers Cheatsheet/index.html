HTML5: Headers

Elements

Tag Name Description
<h1> Header 1 The biggest header, used in most websites.
<h2> Header 2 The second biggest header, used for headings
<h3> Header 3 The third biggest header, used for subheadings.
<h4> Header 4 A small header used for specific information.
<h5> Header 5 The second smallest header, rarely used.
<h6> Header 6 The smallest header, rarely used.

Attributes

The h1-h6 elements use the HTML Global Attributes
